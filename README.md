# HowToDeploy

## 项目概述
自动化部署工具集合，包含多种基础设施的一键部署方案

## 包含组件
- 🚀 AWX自动化部署
- 📊 Zabbix监控系统部署
- ☁️ PackStack OpenStack私有云部署

## 使用说明
1. 进入具体组件目录查看部署指南
2. 所有脚本均需在CentOS Stream 9环境运行
3. 执行前请仔细阅读各组件README中的注意事项
